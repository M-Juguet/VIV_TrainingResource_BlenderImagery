# VIV Website — UI Kit

Recreation of key sections from `https://viv-prod.com` as reusable React components. No build step — load the root `index.html` to see the click-through prototype.

## Components

| File | What it is |
|---|---|
| `Nav.jsx` | Top navigation with logo, menu links, Contact CTA. Goes frosted on scroll. |
| `Hero.jsx` | Black hero with large serif headline and CTA pair. |
| `TalentCategories.jsx` | Three editorial image+copy cards (Design, Digital, SI). |
| `Sectors.jsx` | Six-up photo grid for industry sectors (Transport, Luxe, etc). |
| `Engagements.jsx` | Icon-led 4-column "what we do" block. |
| `ContractTypes.jsx` | Chip row for CDD / CDI / Portage / Freelance / Intermittent. |
| `SecureBand.jsx` | Dark band introducing the "Secure" centralisation offering. |
| `Certifications.jsx` | EcoVadis / ISO 9001 / Qualiopi logos band. |
| `Footer.jsx` | Ink footer with column menus. |

Written in plain JSX loaded via Babel — open `index.html` directly.
